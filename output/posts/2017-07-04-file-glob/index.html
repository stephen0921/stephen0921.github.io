    perl中的glob函数

    Yao He 2017-07-04 16:53

    glob帮助我们快速操作文件

    当我们想对某目录下的文件做操作的时候，使用glob函数是非常方便的。 比如我们想把当前路径的test目录下的所有以.txt结尾的文件列出来： 

    @files = glob './test/*.txt';
print @files;

    或者我们可以这样隐式的写法： 

    while (<'./test/*.txt'>) {
    print "$_\n";
}

    再也不要自己去开发轮子，处理满足某一个条件的多个文件了，Yeah！
